1 . Difference Between Block-level and Inline Elements

Feature Block-level Elements Inline Elements
Definition Elements that takes the full width available. Elements that takes only as much width as necessary.
Width Behavior Expands to fill the entire available width Only takes up as much width as its content
Common Examples <div>, <p>, <h1> to <h6>, <ul>, <li>, <table> <section> <span>, <a>,<em>, <label>, <img>, <button>, <b>, <i>
Example <div>This is a block element</div> <span>This is an inline element</span>



2 . What is the purpose of the alt attribute in an <img> tag?

⇒ It provides alternative text for screen readers and when the image fails to load.




3 . What are semantic HTML elements ?

⇒ Semantic HTML refers to HTML elements that have meaningful names describing their purpose in a webpage.
